# Pruebas E2E (Playwright)

## Requisitos

- Node + pnpm instalados.

## Instalación

1) Instala dependencias:
- pnpm install

2) Instala navegadores:
- pnpm test:e2e:install

## Ejecutar pruebas

- pnpm test:e2e
- pnpm test:e2e:ui

## Notas

- El servidor de Vite se levanta automáticamente para los tests.
- Base URL: http://localhost:5173
